# OIDC-Exchange Serverless Framework Example

Demonstrates deploying oidc-exchange as an AWS Lambda function behind API Gateway using the Serverless Framework.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Deploy:

   ```bash
   npx serverless deploy
   ```

The OIDC-Exchange endpoints are available under `/auth/*` on the generated API Gateway URL.
